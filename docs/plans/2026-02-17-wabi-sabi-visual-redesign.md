# 磨豆机刻度转换器 - 侘寂视觉重设计

> 日期：2026-02-17
> 范围：仅视觉层，保持现有交互流程和功能不变
> 风格：日式侘寂美学 + 素色系

---

## 设计理念

像一件白瓷杯——安静、留白、少即是多。用空间代替线条，用克制的色彩传递温度。

---

## 2. 字体与排版

### 字体

- 标题 + 结果数值：Noto Serif SC (600/700)，Google Fonts 加载
- 正文/UI：系统字体栈
- 加载策略：`font-display: swap`，只加载 600 和 700 字重

### 排版

| 元素 | 规格 |
|------|------|
| 标题 h1 | 26px / Noto Serif SC 600 / 字间距 0.08em |
| 副标题 | 13px / 字间距 0.15em |
| 卡片标题 | 12px / 400 / 全大写 / 字间距 0.12em |
| 结果数值 | 52px / Noto Serif SC 700 / 赤土色 |
| 说明文字 | 13px / 行高 1.8 |

---

## 3. 卡片与布局

### 卡片

- 无边框（`border: none`）
- 圆角 `6px`
- 阴影 `0 1px 3px rgba(45,42,38,0.06)`
- 内边距 `28px 24px`
- 间距 `20px`

### 整体布局

- 容器最大宽 `440px`
- body padding `24px`
- 标题区 margin-bottom `40px`
- 分隔用留白代替线条

### 输入框与下拉

- 圆角 `4px`
- 焦点色：抹茶绿 `#6b7c5e`
- 焦点阴影：`0 0 0 3px rgba(107,124,94,0.12)`

### 冲煮推荐条

- 背景：`rgba(107,124,94,0.08)`
- 文字：抹茶绿 `#6b7c5e`
- 圆角 `4px`

### Disclaimer

- 无边框无背景，纯淡灰文字

---

## 4. 交互元素

### 语言切换

- 纯文字形态：`中文 · EN`
- 选中态：文字加粗 + 墨色
- 未选中：淡灰色
- hover：下划线

### 方向箭头

- 墨色 `#2d2a26`，非蓝色

### 警告条

- 背景 `rgba(184,134,11,0.08)`
- 文字 `#8a7440`
- 圆角 `4px`

### K-Ultra 记号

- 颜色改为次要文字色 `#8a8278`
- 字重 400

- 所有功能逻辑（转换算法、冲煮推荐、i18n）

- HTML 结构
